# MainSwiper Server Guide

Panduan untuk menjalankan server lokal aplikasi MainSwiper.

## Cara Menjalankan Server

1. **Jalankan Server Python**
   Buka terminal di direktori utama proyek, kemudian jalankan perintah berikut:
   ```bash
   python server.py
   ```

2. **Akses Aplikasi**
   Setelah server berjalan, buka browser web Anda dan akses link berikut:
   [http://localhost:8000/login.html](http://localhost:8000/login.html)
